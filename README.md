
# Url shortner using nodejs
‘Url shortner’ is a web-based application where user can shorten long urls.

## Developed using 
> Languages:<br>
  > 1. HTML
  > 2. CSS
  > 3. JAVASCRIPT<br>
 
> Framework:<br>
  > 1. Node.js<br>
   >> Express js

> Database:<br>
  > 1. MySQL<br>
 
## Features
* Login
* Signup
* Users url history
